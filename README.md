# Analysis of Algorithm Project

This is a project where the simplex algorithm was used the find the maximum amount of profit that can be made given an number of constraints.

## Usage
To run this project, you will need to have:
- Java installed and added to your environment variable

To run:
- Download the project
- Open command prompt and navigate to the dist folder
- Type the following command.

```bash
java -jar AnalysisOfAlgoritmProject.jar
```

... and you should be fine.
